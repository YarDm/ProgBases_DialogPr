Экзаменационное задание Основы программирования

Задание состоит из двух программ, которые должны быть реализованы на любых изучаемых в курсе двух языках программирования, но различных ( например, 1-я программа на Java с JDK, а 2-я на C# с .NET Framework).

Первая программа. - JAVA

В диалоговом приложении на форме разместить элементы управления: список, таблица и многострочное текстовое поле, а также поля ввода и кнопки для выполнения следующих действий:
  • установить указанное количество строк и столбцов в таблице;
  • заполнить таблицу производными значениями в указанном диапазоне;
  • ввести заданное в поле ввода значение в указанную ячейку таблицы;
  • отформатировать текст в таблице шрифтом, выбрав его из заданного списка;
  • отформатировать текст в таблице шрифтом, выбрав его из стандартного диалога выбора шрифта;
  • установить для элемента списка с указанным номером цвет фона, выбрав его из стандартного диалога выбора цвета;
  • установить высоту каждого элемента списка на 5 пунктов больше, чем предыдущего;
  • загрузить файл, выбранный с применением стандартного диалога Открыть файл в многострочное текстовое поле. Руководство по FileChoiser. Сам файл откроется в многострочном текстовом поле в новом окне.

Вторая программа. - С#

Создать приложение с MDI-интерфейсом. В главном окне разместить меню, которое позволяет:
  • открывать новые дочерние окна;
  • закрывать активное дочернее окно;
  • закрывать все дочерние окна;
  • выбирать список элементов управления, которые будут автоматически последовательно добавлены и отображены в активном дочернем окне;
  • устанавливать шрифт для отображения содержимого в элементе управления, имеющем фокус, в активном дочернем окне;
  • отображать в отдельном окне список всех элементов управления, которые были в активном дочернем окне на момент выполнения команды меню;
  • выполнить команды стандартного меню: Копировать, Вставить, Выполнить ... .

Задание сдается в виде файла в формате Word (без архивирования) с кодом программ и скриншотами.
